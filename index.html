CodePark — Collaborative Real-Time Browser-Based IDE

A high-performance, real-time collaborative development environment running entirely inside the web browser.

Key Features

  • Real-time Collaboration: Instant cursor tracking, multiplayer editing using Yjs CRDTs, presence metrics, and built-in chat.
  • Interactive Terminals: A functional xterm.js terminal emulator with built-in commands and Python/NodeJS command runners.
  • Git Integration: Stage, commit, push, create branches, and view git history and file diffs completely client-side.
  • In-Browser Live Execution: Test and run Python or Javascript scripts directly within sandboxed virtual sessions.
  • Social Graph & Devlogs: Connect with teammates, publish project releases, and check out developer achievements.

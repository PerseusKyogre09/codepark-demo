## Highlights

- Initial CodePark closed beta
- New collaborative editor experience
- Multi-user workspaces with presence awareness
- Documentation website and developer guides
- AI assistant for project-aware help
- ContextBase project knowledge base
- Pair programming workflows
- Workspace and project management

## Added

- Cloud-based development environments that start from a project or template
- Real-time collaboration inside the editor
- Live cursors and shared workspace presence
- Comments and discussion workflows for team feedback
- AI assistant access directly from the editor
- ContextBase indexing for codebase-aware answers
- Project creation, import, and settings controls
- Public documentation for getting started, environments, collaboration, AI, and templates
- GitHub integration for repository import flows

## Improved

- Cleaner onboarding flow for first-time users
- More readable empty states and layout spacing across the product
- Faster access to the main dashboard and workspace entry points
- More consistent navigation across the marketing site and app shell

## Fixed

- Fixed release-note navigation so the changelog can expand without manual updates
- Fixed several documentation discovery and build-time consistency issues
- Improved route handling for changelog links and release history

## Known Issues

- Some beta workflows are still being refined for larger teams
- Release notes are still limited to the first public beta release
- Environment and collaboration edge cases may still appear during heavy concurrent use

## Upgrade Notes

- No upgrade path is required for this release
- Existing workspaces continue to load through the same dashboard entry points
- New releases will appear here automatically as Markdown files are added to `src/changelog/`

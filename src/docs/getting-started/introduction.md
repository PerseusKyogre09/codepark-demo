# Introduction

**CodePark** is a collaborative cloud IDE built for developers who code together. It gives every team a shared environment — same editor, same terminal, same context — with zero setup and no screen-sharing required.

## What is CodePark?

CodePark runs your development environment entirely in the cloud. You get a full Linux container with internet access, a real terminal, a Monaco-based code editor with live collaboration, and an AI assistant — all accessible from your browser in seconds.

Whether you're onboarding a new hire, doing a pair-programming session, or reviewing a pull request with a teammate, CodePark puts everyone in the same room.

## Who is it for?

- **Engineering teams** who want seamless remote collaboration without VPN friction
- **Solo developers** who want their environment available from any machine
- **Open-source maintainers** reviewing contributions with contributors in real time
- **Educators and bootcamps** running live coding sessions

## Why CodePark?

Most collaboration tools treat code as a second-class citizen. Screen sharing adds lag. Cloud IDEs don't support real multiplayer. Code review tools don't let you run the code.

CodePark solves all three:

- **Zero lag** — direct WebSocket synchronization via Yjs CRDTs, not screen mirroring
- **Real execution** — your code runs in a sandboxed Linux container, not a browser sandbox
- **Real multiplayer** — multiple cursors, shared terminals, live presence, all in one tab

## Key features

- **Live code editing** — multiple cursors, selections, and awareness in real time
- **Shared terminal** — optional per-session shared terminal for watching builds together
- **AI assistant** — context-aware completions and chat powered by ContextBase
- **Project workspaces** — persistent environments that survive browser closes
- **Guest playground** — share a link for instant access, no account required
- **Pro tier** — longer sessions, private projects, and priority execution queues

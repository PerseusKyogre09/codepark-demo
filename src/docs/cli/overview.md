# CLI Overview

CodePark does not currently ship a public command-line interface.

## What that means

You cannot install an official `codepark` binary today, and the product does not expose a supported CLI workflow for project creation, environment management, or collaboration.

## What to use instead

Use the web app and REST API for current workflows:

- Create and manage projects in the dashboard
- Configure environments with `codepark.toml`
- Import repositories through the GitHub integration
- Use the API for programmatic access

## Common mistake

Do not rely on `codepark ...` commands you see in older notes or examples. Those commands are not implemented as a supported product feature at this time.

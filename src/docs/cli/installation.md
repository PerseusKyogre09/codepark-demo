# Installation

There is no official CodePark CLI installer today.

## Why this page exists

Some docs and development notes mention CLI commands as placeholders. Those are not part of the current product surface.

## Current alternatives

If you need to work with CodePark today:

1. Use the dashboard to create and open projects.
2. Use the editor terminal for project-local commands.
3. Use the REST API for programmatic operations.

## Best practice

If you are building internal tooling around CodePark, target the REST API instead of a non-existent CLI.

# Collaboration Issues

Use this guide when teammates cannot see each other, cannot edit, or are missing from the live session.

## Symptoms

- Collaborator cursors are missing
- Chat messages do not appear
- A teammate cannot edit files
- The session shows only one participant

## Possible causes

- Different project sessions
- Permissions mismatch
- Network disconnect
- The collaborator is in a waiting-room state

## Step-by-step fixes

1. Confirm everyone opened the same project.
2. Check each collaborator's role in project settings.
3. Refresh the page if the session looks stale.
4. Rejoin the project session if needed.
5. Verify the live participants list.

## Prevention tips

- Set roles before the session starts.
- Use the invite flow from the editor, not a copied stale link.
- Keep the project open while collaborators are joining.

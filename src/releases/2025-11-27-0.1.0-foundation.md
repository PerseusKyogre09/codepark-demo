## Highlights

- Collaborative editor prototype lets two people pair in the same buffer with presence cursors.
- Lightweight command palette exposes session invites, theme toggles, and quick repository swaps.
- Added VS Code bridge so early adopters can keep their locals while testing CodePark hosting.

## Fixes

- Resolved an auth redirect loop when signing in with Microsoft enterprise tenants.
- Squashed a race where terminal streaming froze after detaching from a session host.
- Patched the onboarding checklist getting stuck when the tutorial repository import failed.
